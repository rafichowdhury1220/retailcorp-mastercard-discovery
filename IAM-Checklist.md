# IAM Security & Compliance Checklist

- [ ] Identity federation configured (Azure AD / Okta)
- [ ] MFA enabled for privileged roles
- [ ] Service accounts scoped to least-privilege
- [ ] Data access policies enforced in EDP
- [ ] Access reviews scheduled and tracked
- [ ] Audit logs shipped to SIEM and monitored
- [ ] Secrets rotation automated
- [ ] Incident response runbook exists
- [ ] PII masking for Mastercard enrollment and transaction fields
- [ ] Control evidence collected for compliance audits
